# Getting Started — TraceMap

## Prerequisites
- Docker & Docker Compose
- Node.js 20+ and/or Python 3.11+ (depending on the component)
- API keys for any cloud services used (see `.env`)

## Installation
```bash
git clone https://github.com/Kimosabey/trace-map.git
cd trace-map
docker compose up
```
